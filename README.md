# RecipeShopper

## 📌Overview

This Flutter app helps users manage their recipes and generate shopping lists based on selected recipes. Recipes are stored both locally and remotely, and users can view full details including ingredients, images, and instructions.

## ✨ Features

  - 📖 Save & View Recipes (Name, Ingredients, Image, Instructions)
  - 🛒 Generate Shopping Lists from selected recipes
  - 💾 Local Storage using Hive
  - ☁️ Remote Storage using AppWrite (Database & Image Storage)
  - 🏗 MVVM Architecture with GetIt for Dependency Injection
  - 🎨 Resource Management using Spider

## 🔮 Future Enhancements

   - 🔑 User Authentication (Sync recipes across devices & households)
   - 🏠 Household Syncing (Share recipes & shopping lists with family members)
   - 📊 Supply Tracking (Track home inventory & auto-update shopping lists)
   - 📷 Barcode Scanning for easy ingredient tracking
   - 📶 Offline Mode (Sync data when online)

## 🛠️ Tech Stack

   - 🖥 Flutter (Dart)
   - 📦 Hive (Local database)
   - ☁️ AppWrite (Remote database & image storage)
   - 🏗 MVVM + GetIt (State management)
   - 🎨 Spider (Resource management)
